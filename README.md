# NecessaryAdminTool v1.2602.0.0

**Enterprise Active Directory and Remote Management Tool**

Professional IT administration suite for managing Windows environments, Active Directory, and remote systems.

---

## 🚀 Features

### Dashboard & Analytics
- Fleet health overview with real-time statistics
- OS distribution charts (Windows 11, 10, Server 2022, etc.)
- Health score calculation based on online status and OS modernity
- Critical alerts for legacy systems (Windows 7, outdated versions)
- Top 10 computers by uptime ranking

### Automated Remediation
One-click fixes for common IT issues:
- Restart Windows Update service
- Clear DNS cache
- Restart Print Spooler
- Enable WinRM for remote management
- Fix time synchronization
- Clear Event Logs

### Asset Management
- Real-time Active Directory computer scanning
- Advanced filtering and search
- Asset tagging system with auto-tagging rules
- Bookmarks and favorites
- CSV export/import

### Remote Management
Integrated support for:
- TeamViewer, AnyDesk, ScreenConnect
- RemotePC, Dameware, ManageEngine

---

## 📋 Requirements

- **OS:** Windows 10/11 or Windows Server 2019+
- **.NET Framework:** 4.8.1
- **Permissions:** Domain admin or equivalent AD read access

---

## 📦 Installation

1. Download the latest release
2. Extract and run `NecessaryAdminTool.exe`
