## 🚨 Known Limitations (Not Blockers)

### **By Design:**
1. **SQLite Not Enabled** - Requires additional NuGet package
   - User sees helpful error message
   - Can choose CSV/JSON/Access instead

2. **Remote Control Tools Disabled by Default** - Security-first
   - Users must explicitly enable in Options
   - Credentials required for each platform

3. **Auto-Update Requires Internet** - Expected behavior
   - Air-gapped deployments use marker file
   - Manual update via MSI still works
